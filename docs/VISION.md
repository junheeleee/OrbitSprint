# Lumen Run Vision

## Core Identity
Lumen Run is a one-touch sci-fi orbit runner built around fast decisions, readable danger, satisfying skill growth, and a strong neon space identity.

## Player Promise
- The player always feels that survival depends on control, timing, and pattern reading rather than luck.
- Runs should become more intense over time through speed, obstacle patterns, fever states, and escalating rewards.
- Audio, visual effects, title screen, items, and feedback should feel like one unified game world.

## Design Pillars
- Simple controls, deep mastery
- Fair obstacle readability
- Fast restart and strong run-to-run momentum
- Clear audiovisual feedback
- App Store-ready polish
