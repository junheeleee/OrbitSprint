# Roadmap

## Immediate

- Gangnam Dream을 별도 GitHub repository로 분리
- Godot에서 실제 실행 화면 확인 후 UI 클릭/레이아웃 QA
- Gangnam Dream 이벤트 결과 문구 빈 값 보강
- Lumen Run 실제 iPhone 플레이 QA 재진행

## Lumen Run

- 피버 모드 연출 강화
- 실드 획득/소모 시각 효과 강화
- 배경음과 효과음 믹싱 정리
- App Store용 스크린샷, 아이콘, 설명문 준비
- TestFlight 배포 준비

## Gangnam Dream

- 저장/불러오기 실제 플레이 테스트
- 투자 화면 매수 금액 선택 추가
- 이벤트 선택 결과 팝업 추가
- 관계 이벤트의 캐릭터 지속성 강화
- 뉴스와 투자 가격 변동의 연결성 강화
- 엔딩 화면 연출 개선
- 메타 진행도 화면 추가
- GitHub Issues로 콘텐츠/버그/개선 작업 분리

## Documentation

- 주요 변경 후 `docs/WORK_LOG.md` 업데이트
- 큰 설계 변경은 `docs/DECISIONS.md`에 남김
- 출시 준비 단계에서 `CHANGELOG.md`와 릴리즈 노트 추가
