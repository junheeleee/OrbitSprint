# Lumen Run Roadmap

## Prototype
- Core orbit movement
- Collision and scoring
- Basic audio
- Simulator and device testing

## Playable Alpha
- Balanced obstacle spawning
- Three-depth orbit behavior
- Fever mode
- Shield item feedback
- Start/loading screens
- Improved post-game results flow
- Pattern-based stage pacing so runs stop feeling identical
- Distinct obstacle waves, safe-lane windows, and risk/reward item routes

## Polish Beta
- First beta plan and playtest checklist
- Stronger sci-fi identity
- Better music and sound direction
- Visual effects pass
- Difficulty pacing
- Localization review
- Game Center leaderboard
- Achievement set
- Unlockable cosmetic rewards
- Daily/weekly challenge structure
- Dedicated rewards screen, with store mechanics considered only after an in-game currency exists

## Release Candidate
- App icon and screenshots
- Final device testing
- App Store metadata
- Privacy and signing checks
- Game Center availability fallback
- Release notes and QA checklist pass
