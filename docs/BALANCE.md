# Lumen Run Balance

Track gameplay tuning changes here so the game becomes more skillful and less random over time.

## Core Variables To Track
- Base player speed.
- Speed increase per score/time.
- Orbit transition duration.
- Obstacle spawn interval.
- Obstacle travel speed.
- Safe spawn angle rules.
- Respawn invulnerability duration.
- Shield duration.
- Fever charge rate.
- Fever duration.
- Fever score multiplier.
- Item spawn probability.

## Current Balance Goals
- Death should feel avoidable with good timing.
- Difficulty should increase gradually.
- Respawn should always give the player a fair recovery window.
- Fever should create a short, memorable power spike.
- Shield should be valuable but not remove all tension.
