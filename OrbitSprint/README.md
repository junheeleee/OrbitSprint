# Orbit Sprint

Orbit Sprint is a one-touch iOS arcade game prototype built with SwiftUI and SpriteKit.

Tap to switch between the inner and outer orbit. Collect yellow sparks, dodge red shards, and chase a persistent best score.

## Open

Open `OrbitSprint.xcodeproj` in Xcode, choose an iPhone simulator, then run the `OrbitSprint` scheme.

## App Store Prep

- Replace the placeholder app icon with a 1024x1024 production icon.
- Set a unique bundle identifier in Xcode.
- Add screenshots, privacy details, and age rating metadata in App Store Connect.
- Test on physical devices before submission.
