# Lumen Run

Lumen Run is a one-touch iOS neon arcade game built with SwiftUI and SpriteKit.

Tap to shift through three orbit depths. Collect lumen sparks, dodge glitch shards, trigger Fever Sync, and chase a persistent best score.

## Open

Open `LumenRun.xcodeproj` in Xcode, choose an iPhone simulator, then run the `LumenRun` scheme.

## App Store Prep

- Replace the placeholder app icon with a 1024x1024 production icon.
- Set a unique bundle identifier in Xcode.
- Add screenshots, privacy details, and age rating metadata in App Store Connect.
- Test on physical devices before submission.
